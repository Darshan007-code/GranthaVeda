**GranthaVeda** is a premium, full-stack book discovery platform inspired by the rich heritage of **Indian Sanskriti**. Designed for true seekers of knowledge, it combines a modern "Royal Sanskriti" aesthetic with powerful search capabilities across millions of global titles.

---

## ✨ Key Features

### 🏛️ Royal Sanskriti UI
- **Permanent Dark Mode:** A deep, focused "Sandalwood & Stone" aesthetic.
- **Glassmorphism:** Premium blurred navigation and interactive components.
- **Animations:** Fluid transitions powered by **Framer Motion**.

### 🔍 Knowledge Discovery
- **Multi-Source Engine:** Unified search across **Google Books** and **Open Library**.
- **Spheres of Wisdom:** Curated genre-based navigation (Philosophy, Science, History, etc.).
- **"Read Now" Access:** Direct links to full digital copies for public domain books.

### 👥 Seeker Ecosystem
- **User Accounts:** Personalized dashboards for every seeker.
- **Reading Tracker:** Visual progress bars to track your intellectual journey.
- **XP System:** Earn "Knowledge Points" based on your reading activity.

### 📊 Vault Overlord (Admin)
- **Governance Dashboard:** Complete control panel for administrators.
- **Seeker Registry:** Monitor and manage the entire community of readers.
- **Ecosystem Analytics:** Live status of the knowledge vault.

## 🛠️ Tech Stack

**Frontend:**
- React (Vite + TypeScript)
- Tailwind CSS v4
- Framer Motion
- Swiper.js (Carousel)
- Lucide React (Icons)

**Backend:**
- Node.js & Express
- MongoDB (Mongoose)
- JWT (JSON Web Tokens) for Auth
- Bcrypt.js (Security)

## 🚀 Getting Started

### 1. Prerequisites
- Node.js installed
- MongoDB installed and running locally

### 2. Installation
```bash
# Clone the repository
git clone https://github.com/Darshan007-code/book-finder.git

# Setup Backend
cd book-finder/backend
npm install
node seed.js  # Seed the database with default accounts
node server.js

# Setup Frontend
cd ../frontend
npm install
npm run dev
```
